# FinEdu KZ — Финансовая грамотность для школьников

Готовый адаптивный образовательный сайт для учеников Республики Казахстан.

## Стек

- HTML5
- CSS3 (glassmorphism, адаптив, анимации)
- Vanilla JavaScript (интерактив, тесты, калькуляторы, прогресс)
- Без серверной части (полностью GitHub Pages compatible)

## Структура проекта

```text
index.files/
├─ index.html
├─ style.css
├─ script.js
├─ README.md
└─ assets/
   ├─ images/
   │  └─ .gitkeep
   └─ icons/
      └─ .gitkeep
```

## Запуск локально

1. Скачайте или склонируйте проект.
2. Откройте файл `index.html` в браузере.
3. Готово: сайт полностью работает локально без сервера.

## Как создать GitHub-репозиторий

1. Откройте [GitHub](https://github.com) и нажмите **New repository**.
2. Укажите имя, например: `finedu-kz`.
3. Выберите **Public** (для GitHub Pages).
4. Нажмите **Create repository**.

## Как загрузить файлы в репозиторий

### Вариант A (через интерфейс GitHub)

1. Откройте созданный репозиторий.
2. Нажмите **Add file** → **Upload files**.
3. Перетащите все файлы и папки проекта.
4. Нажмите **Commit changes**.

### Вариант B (через Git в терминале)

```bash
git init
git add .
git commit -m "Initial commit: FinEdu KZ website"
git branch -M main
git remote add origin https://github.com/<ВАШ_ЛОГИН>/<ИМЯ_РЕПО>.git
git push -u origin main
```

## Как включить GitHub Pages

1. Откройте репозиторий на GitHub.
2. Перейдите в **Settings** → **Pages**.
3. В разделе **Build and deployment** выберите:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. Нажмите **Save**.
5. Через 1-3 минуты появится ссылка вида:
   `https://<логин>.github.io/<имя-репозитория>/`

## Как обновлять сайт

1. Измените нужные файлы локально.
2. Сохраните изменения.
3. Загрузите изменения в GitHub:

```bash
git add .
git commit -m "Update content and tasks"
git push
```

GitHub Pages автоматически обновит сайт.

## Как менять тексты

- Откройте `index.html` для статичных заголовков и секций.
- Откройте `script.js` для динамических текстов:
  - массив `courses`
  - массив `worksheets`
  - массив `practiceCategories`
  - массив `quizBank`
  - массив `dailyTasks`

## Как добавлять новые задания

Откройте `script.js` и добавьте объект в `practiceCategories`:

```js
{
  name: "Новая категория",
  level: "Средний",
  q: "Текст вопроса",
  options: ["Вариант 1", "Вариант 2", "Вариант 3"],
  answer: 1
}
```

`answer` — индекс правильного варианта (нумерация с 0).

## Как обновлять тесты

Откройте `script.js` и расширьте `quizBank`:

```js
{
  q: "Новый вопрос",
  options: ["A", "B", "C"],
  answer: 0
}
```

## Как менять изображения и иконки

1. Добавьте файлы в:
   - `assets/images/`
   - `assets/icons/`
2. Подключайте их в `index.html` или через CSS (`style.css`).

Пример:

```html
<img src="assets/images/finance-banner.png" alt="Финансовое обучение">
```
